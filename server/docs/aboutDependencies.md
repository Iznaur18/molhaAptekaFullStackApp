# Зависимости сервера

**express** — веб-фреймворк: маршруты, middleware, обработка HTTP-запросов и ответов.

**mongoose** — ODM для MongoDB: схемы, модели, запросы к БД, валидация на уровне данных.

**cors** — разрешает запросы с других доменов/портов (нужно для фронта на отдельном origin).

**bcrypt** — хеширование паролей (безопасное хранение, проверка при логине).

**jsonwebtoken** — создание и проверка JWT: аутентификация и авторизация по токенам.

**express-validator** — валидация и санитизация входящих данных (body, query, params).

**multer** — загрузка файлов (multipart/form-data): сохранение на диск или в память.

**nodemon** — перезапуск сервера при изменении файлов (удобно для разработки).

**Модули (import/export)** - Синтаксис import — это ES-модули. Чтобы так работало, в package.json этого проекта должно быть "type": "module".
