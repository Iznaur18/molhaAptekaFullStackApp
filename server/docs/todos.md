Товары (препараты)	Модель Product (название, описание, цена, остаток, категория). CRUD, поиск, фильтры. Связь с рейтингом/отзывами при желании.

Корзина	Модель Cart (user, items: productId + quantity). Endpoints: добавить/удалить/получить корзину.

Заказы	Модель Order (user, items, сумма, адрес, статус). Создание заказа из корзины, история заказов, смена статуса (в т.ч. для админа/фармацевта).

Поиск пользователей	GET /user?search=...&role=pharmacist с пагинацией — для админки или выбора фармацевта.

Список пользователей для админа	GET /user с фильтрами (роль, активность, блок) и пагинацией, только для роли admin.
